🔐 Hashing
GET /hash/algorithms List all supported algorithms ▼

Returns a list of all cryptographic hashing algorithms supported by this API.

GET /hash/{algorithm}/{text} Hash any string ▼

Hash any string using your chosen algorithm. Great for checksums, verifying integrity, or learning how hashing works.

🔄 Encoding
GET /encode/{method}/{text} Encode a string ▼

Encode any string using Base64, Hex, or URL encoding.

GET /decode/{method}/{encoded} Decode a string ▼

Decode a Base64, Hex, or URL-encoded string back to its original form.

🌐 Network
GET /ip/{address} IP info + geolocation ▼

Get info about any IPv4 or IPv6 address — location, org, timezone, type. Pass me to look up your own IP.

GET /time Current UTC time ▼

Returns the current UTC time in multiple formats: ISO, Unix timestamp, Unix ms, date, and time.

🔑 Password
POST /password/analyze Analyze password strength ▼

Scores a password on 10 security checks and returns strength label, entropy estimate, and actionable feedback. ⚠ Don't use a real password here.

🛡️ Security Tools
GET /recon Domain recon — IP, headers, tech stack, security gaps ▼

Resolves IP, probes HTTP headers, detects technologies, identifies missing security headers, and generates a plain-English smart_summary.

GET /analyze-url Follow redirects, detect misconfigurations ▼

Follows redirect chain, inspects final headers, flags misconfigs like CORS wildcards, exposed server banners, and missing HSTS.

GET /bb-scan Bug bounty recon — probe common sensitive paths ▼

Probes 25+ paths: /admin, /.env, /api/v1, /.git/config and more. Returns non-404 hits and bug bounty hints. Only test targets you have permission to test.

GET /payloads Categorized attack payloads for security testing ▼

Returns curated payloads for testing. Use only on systems you own or have explicit permission to test.
